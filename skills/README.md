# DeepWork Pi Skills

This directory is declared as the package skill root in `package.json`.

Native Pi skill files should be added here following the requirements in `doc/specs/deepwork-pi/PI-REQ-001-native-pi-extension.md`. Required skill names include:

- `deepwork`
- `review`
- `configure-reviews`
- `deepreviews`
- `deepschema`
- `deepplan`
- `new-user`
- `record`

Skill instructions must describe native Pi tools and commands, not MCP tools.
